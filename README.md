# 📖 About the Project

The **ATM Interface** is a console-based banking application developed using **Java**. It simulates the core operations of an Automated Teller Machine (ATM) and demonstrates the practical implementation of **Object-Oriented Programming (OOP)** concepts.

The project provides a simple and interactive menu-driven interface where users can securely perform banking operations.
